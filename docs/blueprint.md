# **App Name**: FlavorVault

## Core Features:

- Ingredient Management: Add ingredients to a personal pantry list via an input field and button, and remove items with a 'Delete' action for each. All data will use local mock data.
- AI Recipe Suggestion Tool: A prominent button to trigger the generative AI tool to suggest a complete recipe based on the ingredients currently listed in the user's pantry.
- Recipe Card Display: Render a visually distinct recipe card displaying a mock generated recipe, including its title, prep time, required ingredients, and 3-step instructions.
- Save Recipe Action: A 'Save to Recipe Box' button to indicate where generated recipes could eventually be stored (initially for visual interaction only).
- Responsive Layout: Implement a two-column responsive layout: 'My Pantry' sidebar (1/3 width) and 'Kitchen Workspace' main area (2/3 width).

## Style Guidelines:

- Color scheme: Light. Inspired by warm, food-friendly tones like soft oranges and off-whites. Primary brand color: A warm, inviting orange (#EB8644). Background color: A subtle, desaturated off-white with a hint of orange (#F8F1EE) to provide a soft canvas. Accent color: A vibrant, clear red (#F22A3A) to draw attention to call-to-action buttons and important highlights, providing good contrast.
- Body and headline font: 'PT Sans' (humanist sans-serif) for its modern yet approachable feel, suitable for both titles and legible body text, reinforcing the clean and friendly aesthetic.
- Use simple, line-style icons for actions like 'Add Ingredient' and 'Delete' to maintain a clean, modern, and minimalist look. Food-related icons can be used where appropriate.
- The application will feature a two-column structure; a fixed 1/3 width sidebar on the left for 'My Pantry' and a fluid 2/3 width main section on the right for the 'Kitchen Workspace'. This layout will adapt gracefully on smaller screens, potentially stacking columns vertically.
- Incorporate subtle animations for user interactions, such as a gentle fade-in when new ingredients are added or a slight hover effect on buttons and recipe cards, to enhance user feedback and overall app responsiveness.
